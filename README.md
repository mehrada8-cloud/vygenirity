# Vygen Signals

پروژه‌ی ساده PHP برای مدیریت و نمایش سیگنال‌ها با بروزرسانی سبک بر اساس ETag.

## نصب سریع
1. سورس را از گیت‌هاب دانلود و روی هاست استخراج کنید.
2. مطمئن شوید DocumentRoot وب‌سرور روی پوشه `public/` تنظیم شده است.
3. فایل `config/app.php` را باز کرده و موارد زیر را تنظیم کنید:
   - `base_url`
   - رمز عبور ادمین (`admin.password`)
   - `price_sources` برای دریافت قیمت‌ها

## آدرس‌های مهم
- صفحه دسته‌بندی‌ها: `/categories`
- صفحه سیگنال‌ها: `/signals/{slug}`
- پنل مدیریت: `/admin/login`

## کرون‌جاب‌ها
دو اسکریپت کرون برای بروزرسانی قیمت و بررسی سیگنال‌ها وجود دارد:

```bash
# دریافت قیمت‌ها و ذخیره در JSON
php cron/fetch_prices.php

# بررسی سیگنال‌ها برای تارگت/استاپ و بروزرسانی پیشرفت
php cron/process_signals.php
```

پیشنهاد برای کرون (هر 1 دقیقه):
```
* * * * * /usr/bin/php /path/to/project/cron/fetch_prices.php
* * * * * /usr/bin/php /path/to/project/cron/process_signals.php
```

## نکات
- فایل قیمت‌ها در `data/prices.json` نگهداری می‌شود.
- اگر نماد جفت ارز در قیمت‌ها نباشد، سیگنال همچنان ثبت می‌شود اما بروزرسانی خودکار ندارد.
- نمایش تاریخ به صورت همزمان میلادی و شمسی انجام می‌شود.
